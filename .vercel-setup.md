# Configuration Vercel

## Étape 1 : Se connecter à Vercel
```bash
vercel login
```

## Étape 2 : Lier le projet au repo GitHub
1. Allez sur https://vercel.com/new
2. Connectez votre compte GitHub si ce n'est pas déjà fait
3. Importez le repo : `Mikawx3/chinese-exam-trainer`
4. Vercel détectera automatiquement Next.js
5. Cliquez sur "Deploy"

## Étape 3 : Configuration automatique
Vercel configurera automatiquement :
- Build Command: `npm run build`
- Output Directory: `.next`
- Install Command: `npm install`

## Étape 4 : Déploiement automatique
Une fois configuré, chaque push sur la branche `main` déclenchera automatiquement un nouveau déploiement.

## URL du projet
Votre application sera disponible à : `https://chinese-exam-trainer.vercel.app` (ou un autre nom si vous le changez)
